# Scripts de Démarrage Rapide

Ce dossier contient des scripts pour faciliter le développement et le déploiement.

## Windows

### `start-dev.ps1`
Démarre l'environnement de développement complet avec Docker Compose

### `start-local.ps1`
Démarre l'API et le frontend en local (sans Docker)

### `install-all.ps1`
Installe toutes les dépendances npm pour l'API et le frontend

### `start-local.sh`
